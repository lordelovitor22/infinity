Resumo das Tags HTML

Estrutura Básica

  • <!DOCTYPE html> - Define o tipo de documento como HTML5. (Não tem fechamento)
  • <html>...</html> - Elemento raiz do documento.
  • <head>...</head> - Contém metadados, como título e links para CSS.
  • <title>...</title> - Define o título que aparece na aba do navegador.
  • <body>...</body> - Contém o conteúdo visível da página.

Tags de Texto e Estrutura

  • <h1>...</h1> a <h6>...</h6> - Cabeçalhos, do maior (<h1>) ao menor (<h6>).
  • <p>...</p> - Define um parágrafo.
  • <br> - Quebra de linha. (Não tem fechamento)
  • <hr> - Linha horizontal para separar conteúdo. (Não tem fechamento)

Formatação de Texto

  • <strong>...</strong> - Destaca texto em negrito (importante).
  • <em>...</em> - Destaca texto em itálico (ênfase).
  • <b>...</b> - Negrito.
  • <i>...</i> - Itálico.

Links e Imagens

  • <a>...</a> - Define um link.
    • href: Especifica a URL de destino.
    • target: Define onde abrir o link (_blank abre em nova aba).
  • <img> - Insere uma imagem. (Não tem fechamento)
    • src: URL da imagem.
    • alt: Texto alternativo que descreve a imagem.
    • width e height: Definem o tamanho da imagem.

Listas

  • <ul>...</ul> - Lista não ordenada.
  • <ol>...</ol> - Lista ordenada.
  • <li>...</li> - Item de lista.

Tabelas

  • <table>...</table> - Define uma tabela.
  • <tr>...</tr> - Define uma linha na tabela.
  • <th>...</th> - Define uma célula de cabeçalho.
  • <td>...</td> - Define uma célula de dados.

Elementos Semânticos

  • <header>...</header> - Representa o cabeçalho de uma página ou seção.
  • <nav>...</nav> - Define uma área de navegação.
  • <main>...</main> - Representa o conteúdo principal da página.
  • <section>...</section> - Define uma seção temática.
  • <article>...</article> - Representa um conteúdo independente.
  • <aside>...</aside> - Define conteúdo relacionado, como barras laterais.
  • <footer>...</footer> - Representa o rodapé da página ou seção.

Elementos Não-Semânticos

  • <div>...</div> - Define uma divisão genérica, sem significado específico.
  • <span>...</span> - Define uma pequena área de texto, usada para estilização ou agrupamento.

Criação de Formulários

  • <form>...</form> - Define um formulário para entrada de dados.
  • <input> - Campo genérico de entrada de dados. (Não tem fechamento)
    • type="text": Campo para entradas curtas.
    • type="password": Campo com conteúdo oculto para senhas.
    • type="email": Campo que valida e-mails.
    • type="url": Campo que valida URLs.
    • type="search": Campo estilizado para pesquisas.
    • type="tel": Campo para números de telefone.
    • type="checkbox": Caixa de seleção para múltiplas escolhas.
    • type="radio": Botão de opção para uma escolha em um grupo.
    • type="range": Controle deslizante para selecionar valores dentro de um intervalo.
    • type="color": Seletor de cor.
    • type="date": Seletor de data.
    • type="datetime-local": Seletor de data e hora sem fuso horário.
    • type="month": Seletor de mês e ano.
    • type="week": Seletor de semana.
    • type="time": Seletor de hora.
    • type="number": Campo para números com botões de incremento e decremento.
    • type="file": Permite que o usuário selecione um ou mais arquivos para upload.
    • type="submit": Envia o formulário.
    • type="reset": Redefine os campos do formulário.
    • type="button": Botão genérico para ações personalizadas com JavaScript.
  • <textarea>...</textarea> - Área de texto para entradas mais longas.
  • <select>...</select> - Cria um menu suspenso.
    • <option>...</option> - Define uma opção dentro do menu.
  • <button>...</button> - Botões com funções específicas, como enviar (type="submit") ou redefinir (type="reset").
